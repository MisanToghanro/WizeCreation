# WizeCreation Portfolio Website

A modern portfolio website built for **WizeCreation**, showcasing branding, graphic design, and visual identity projects through a clean and premium user experience.

## ✨ Overview

This project is a fully responsive portfolio website designed to present creative work, brand identity projects, and visual design services in a modern and engaging way.

The rebrand focused on:

* A premium visual identity
* Modern UI/UX improvements
* Responsive layouts across devices
* Improved gallery presentation
* Updated typography and color system
* Cleaner project storytelling

---

## 🚀 Features

* Responsive modern design
* Dynamic gallery/project pages
* Case study style project details
* Custom 404 page
* Glassmorphism inspired UI
* Brand-focused color system
* Social media integration
* Optimized image handling with Next.js
* Mobile-friendly navigation

---

## 🛠️ Tech Stack

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/MisanToghanro/WizeCreation.git
```

Navigate into the project:

```bash
cd WizeCreation/portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
